# Ссылки
Чтобы использовать ссылку на внутреннюю заметку Обсидиана: 
```
[[Название_файла]]
```
Чтобы сослаться на блок в заметке - используем
```
[[Название_файла^блок|отображаемое_имя_ссылки]]
```
